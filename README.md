# TermRun
Simple tasks runner tool with TUI. Supports hierarchical task placement based on files structure of your tasks folder


# Dependencies
* [Invoke](https://github.com/pyinvoke/invoke) - used as a executor for terminal commands
* [Rich](https://github.com/Textualize/rich) - advanced terminal formatting
